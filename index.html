What is the Documet object model(DOM)?

"The Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document."

The DOM standard is separated into 3 different parts:

1: Core DOM - standard model for all document types, 2: XML DOM - standard model for XML document, 3: HTML DOM - standard model for HTML documents.

How do you select on element from the DOM?

To select an element from the DOM in JavaScript, you can use the following methods

getElementById(): Finds an element based on its unique ID.

getElementsByTagName(): Searches for elements by tag name.

getElementsByClassName(): Searches for elements by class name.

getElementsByName(): Searches for elements by name attribute.

What is event delegation in the context of the DOM , and why is it usefule?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.
